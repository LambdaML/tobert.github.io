64-bit perl on HP-UX (PARISC or Itanium)

2005-10-13 - Here's something that took me a few hours to get dialed in. This is all set for linking DBD::Oracle and DBD::Sybase to their respective 64-bit client libraries without hassle. I'm assuming the HP ANSI C compiler is installed.
